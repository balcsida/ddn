Cloud DB 2.0

The successor to the previous cloud database, this one is much more versatile and extensible.

On the right, you can see the active connectors; These are the available database servers in which you can create and import databases.

Currently, the following database vendors are tested and supported:

  1. MySQL 5.5, 5.6, 5.7
  2. Oracle 11g

There are plans to add support for PostgreSQL and SQL Server as well

Also works with .zip, .tar, .gz and .tar.gz archives
